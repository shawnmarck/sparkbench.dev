Run it on your own Spark.

SparkBench is the operator tool behind this site. It bundles a portal, model inventory, inference control plane, and reproducible benchmark harness — the same one that generates the numbers you're looking at.

  • One CLI — discover, download, switch, bench, promote.
  • Real recipes — vLLM (eugr), llama.cpp, ds4. Auto-scaffolded from weights.
  • Idempotent installers — re-run anything without breaking things.
  • Your data, your box — nothing leaves the LAN unless you ship it back here.
install shell 
# Clone to /opt/spark on your DGX Spark
git clone https://github.com/shawnmarck/sparkbench.git /opt/spark
cd /opt/spark

# Bootstrap CLI, model inventory, inference API
sudo bash install/20-spark-cli.sh
sudo bash install/04-model-inventory.sh
sudo bash install/17-inference-api.sh

# Browse, switch, bench
spark inference list
spark inference up qwen36-q4-llama
spark inference bench
